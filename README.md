# Hypothesis-Driven Exploratory Data Analysis of Streaming Catalog Trends

Executed a comprehensive exploratory data analysis on a custom streaming dataset to uncover historical production patterns and global distribution strategies. Emphasizing traditional statistical techniques and visual storytelling over complex modeling, this study rigorously challenged industry assumptions regarding evolving content durations and delivered actionable, data-driven insights into genre popularity and audience targeting.

### 📂 Dataset

- Custom-built streaming dataset (`mymoviedb.csv`)
- Historical movie and TV show offerings
- Features include content durations, release years, genres, and geographic origins

### 🚀 Pipeline Overview

- Dataset wrangling and data cleaning
- Yearly production trend analysis
- Genre popularity shift tracking
- Content duration hypothesis testing
- Geographic distribution mapping
- Visual storytelling and reporting

### 🛠 Tech Stack

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebooks

### 📌 Key Results

- **Historical Trend Analysis:** Identified historical trends shaping streaming content strategy and production volume.
- **Industry Insight:** Successfully challenged modern industry assumptions regarding viewer behavior and the rise of "shorter content."
- **Strategic Mapping:** Mapped actionable insights on genre popularity shifts and audience targeting.
- **Global Distribution:** Visualized global distribution strategies via country-wise content mapping.
